# 半调网屏与印刷效果生成器版本更新方案

## 项目版本规划目标

本项目采用 **渐进式迭代** 的方式推进。

目标不是一次性做成复杂系统，而是：
- 先完成本地可运行的轻量编辑器骨架
- 再补齐两条主线
- 再增强半调与印刷效果
- 最后优化性能与输出质量

这里的“两条主线”必须在版本规划中持续被保证：

1. **预设后细调**
2. **从头搭建效果**

版本路线不能只优先照顾预设体验，也不能只做纯手工参数实验，必须让两条主线同步成立。

---

## 一、版本规划原则

### 原则 1：先编辑器骨架，后风格堆叠
优先搭起导入、调整、效果、预览、导出的完整链路。

### 原则 2：两条主线都要可用
每个关键版本都要检查：
- 预设后细调是否成立
- 从头搭建效果是否成立

### 原则 3：先单通道和基础印刷风，再做高级模拟
先完成基础黑白 / 单色 / 基础印刷质感，再考虑更复杂的多色与分层。

### 原则 4：预览与导出分离
避免过早把性能压力集中到高分辨率导出上。

---

## 二、版本路线总览

```text
v0.1  编辑器骨架版
v0.2  双主线可用版
v0.3  半调模式增强版
v0.4  印刷效果增强版
v0.5  性能与导出优化版
v0.6+ 高级印刷探索版
v1.0  稳定发布版
```

## 三、详细版本规划

## v0.1 — 编辑器骨架版

### 版本目标
先把“本地轻量编辑器”的基本工作流跑通。

### 核心功能
- 本地图片上传
- Canvas 显示原图
- 基础预览区
- 基础参数面板框架
- 灰度化
- 亮度 / 对比度基础调节
- 本地预览更新

### 本版本重点
- 搭起编辑器框架
- 搭起基础调图链路
- 为后续两条主线做容器准备

### 完成标准
- 用户能上传图片
- 用户能看到调整后的结果
- 参数改动会更新预览
- 页面结构已经具备编辑器形态

---

## v0.2 — 双主线可用版

### 版本目标
让项目正式具备两条完整主线。

### 新增功能
- 预设系统初版
- 默认空白参数初版
- “预设后细调”入口
- “从头搭建效果”入口
- 参数重置
- 示例图片加载

### 对两条主线的要求
#### 主线 A：预设后细调
- 用户选择预设后，参数会被写入编辑器
- 用户能继续修改这些参数

#### 主线 B：从头搭建效果
- 用户可跳过预设
- 用户可直接从默认参数开始搭建
- 参数面板完整可用

### 本版本重点
- 两条主线都能独立完成一次效果生成
- 预设不是黑盒
- 从头搭建不是隐藏功能

### 完成标准
- 用户可按任一主线完成一次编辑
- 两条主线都能到达可导出前的结果

---

## v0.3 — 半调模式增强版

### 版本目标
让编辑器真正具备“半调创造器”能力。

### 新增功能
- 黑白圆点半调
- 方点模式
- 线条模式
- 角度控制
- 反相
- before / after 预览

### 对两条主线的影响
- 预设主线：预设可快速切入不同半调风格
- 从头搭建主线：用户可以自己决定点形、角度和参数映射

### 本版本重点
- 让半调不再只是一个结果，而是可编辑的效果模块
- 增强项目辨识度

### 完成标准
- 至少 3 种半调模式可用
- 两条主线都能使用这些模式
- README 可展示多种结果案例

---

## v0.4 — 印刷效果增强版

### 版本目标
从“半调编辑器”走向“半调 + 印刷效果生成器”。

### 新增功能
- 纸张纹理叠加
- 颗粒噪声
- 基础双色风格
- 轻量复古印刷风预设
- 导出前背景 / 前景色控制

### 对两条主线的影响
- 预设主线：预设可快速形成更完整的印刷风格
- 从头搭建主线：用户可自己叠加纸张、颗粒、双色等效果

### 本版本重点
- 强化“类似 PS 简化版”的感觉
- 让项目不只是生成半调，而是生成印刷风格成品

### 完成标准
- 用户可得到明显区别于单纯半调的印刷风格结果
- 两条主线都能使用印刷效果模块

---

## v0.5 — 性能与导出优化版

### 版本目标
提升本地使用体验和最终输出质量。

### 新增功能
- 低清预览 / 高清导出分离
- PNG 导出
- 可选 JPEG 导出
- 导出倍率设置
- 参数调节防抖
- Web Worker 初步接入

### 本版本重点
- 提升本地大图处理流畅度
- 让导出图更适合实际设计使用

### 完成标准
- 预览流畅度明显提升
- 导出图质量高于预览图
- 项目开始具备“日常工具”而不只是“效果实验”的体验

---

## v0.6 — 高级印刷探索版

### 版本目标
进入更高级的印刷风格扩展阶段。

### 可选功能
- 假 CMYK 分层
- 通道独立角度
- 套印偏移效果
- 更复杂的双色 / 多色印刷风
- 油墨扩散感增强

### 本版本重点
- 扩展视觉表达上限
- 为作品集与高级案例提供更强展示内容

### 完成标准
- 项目明显超出“基础滤镜工具”范围
- 已具备初步的高级印刷风格模拟能力

---

## v1.0 — 首个正式稳定版

### 版本目标
形成完整、可发布、可长期维护的正式版本。

### 建议包含
- 稳定的图片导入
- 两条主线稳定可用
- 基础调图完整
- 半调模块完整
- 印刷效果模块完整
- 本地导出完整
- README 完整
- 示例图完整
- 文档结构清晰
- GitHub 项目形态成熟

### 完成标准
- 普通用户可直接 clone 或本地运行使用
- 两条主线体验清晰
- 功能边界明确
- 可以作为 GitHub 作品或长期维护项目发布

## 四、版本节奏建议

建议采用以下节奏：

### 里程碑 1：两周内
完成 v0.1 + v0.2

### 里程碑 2：一个月内
完成 v0.3

### 里程碑 3：一个半月至两个月
完成 v0.4 + v0.5

### 里程碑 4：之后按兴趣扩展
探索 v0.6 与 v1.0
