# 半调网屏与印刷效果生成器

一个 local-first 的纯前端图像处理工具，目标是完成本地图片导入、基础调图、半调网屏生成、印刷质感叠加和本地导出。

## 当前状态

- 技术栈：React + TypeScript + Vite + Canvas 2D
- 当前阶段：第一版编辑器骨架
- 已具备：图片导入、Canvas 预览、基础调整、半调参数、预设入口、印刷效果参数、本地 PNG 导出

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
